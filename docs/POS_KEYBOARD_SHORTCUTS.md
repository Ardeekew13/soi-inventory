## ğŸ“ Usage Notes

### Park Order (F2)
- Saves current cart items
- Preserves order type and table number
- Works offline - syncs when back online
- Shows kitchen receipt after parking

### Open Payment (F3)
- Opens payment modal
- Quick way to start checkout process
- Validates cart is not empty
- Checks cash drawer is open

### Parked Orders (F4)
- Opens drawer showing all parked orders
- Click to load order back to cart
- Can void/delete parked orders
- Shows both online and offline parked orders

### Clear Cart (ESC)
- Shows confirmation dialog
- Only works when cart has items
- Clears all selections (discount, service charge, table)
- Safe to use - requires confirmation
